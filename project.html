Project Details

Cloud-Based Static Website Hosting on Microsoft Azure

Steps Followed

  1. Created a Resource Group in Azure Portal.
  2. Deployed a new Storage Account with Standard performance and LRS redundancy.
  3. Enabled the Static Website feature, specifying index.html and error.html.
  4. Uploaded all the required files (HTML, CSS, images) to the $web container.
  5. Accessed the site using the Primary Endpoint provided by Azure.
